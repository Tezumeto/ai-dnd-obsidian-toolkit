## Правила работы с файлами

### Изменение файлов
При изменении добавляй после frontmatter:
```markdown
> [!changelog]
> **YYYY-MM-DD ([имя_агента]):** Краткое описание изменений.
```

Храни не более 5 последних записей — старые удаляй.

### Удаление файлов
Не удаляй файлы. Перемещай их в `zАрхив/DELETED/`.

### Создание файлов
Следуй существующей структуре проекта и стилю оформления соседних файлов. Skills (`.llm/skills/`) содержат полные структуры для создания файлов.
