# Инструкция по внедрению оптимизации (v2.0)

Чтобы активировать новую систему контекста, выполни следующие действия:

1.  **Обнови `system.md`:** Замени содержимое `DND/.llm/prompts/system.md` текстом из черновика.
2.  **Обнови бутлоадеры:**
    - Положи `GEMINI.md` в корень проекта (рядом с `.gemini/` или в корень диска, где агент ищет инструкции).
    - Положи `CLAUDE.md` в корень проекта.
3.  **Пометь неактуальное:** Если есть файлы, которые точно устарели, добавь им в Frontmatter `status: stale` или тег `#stale`.
4.  **Следи за именованием Summary:** Убедись, что новые резюме всегда сохраняют формат `{агент}_{YYYY-MM-DD}.md`, чтобы АИ мог легко находить самый свежий файл.

### Что это даст:
- **Экономия токенов:** Агент не будет читать лишнее при старте.
- **Умная навигация:** АИ перестанет путаться в архивных данных.
- **Синхронизация:** Любой новый агент (даже если ты переключишься на другую модель) сразу поймет, где остановился предыдущий.
