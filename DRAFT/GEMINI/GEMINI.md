# Gemini CLI: Val'trey Bootstrapper

Твои инструкции и контекст проекта находятся во внешних файлах. Это необходимо для синхронизации работы разных агентов.

## Твой алгоритм при запуске:
1.  **Прочитай `DND/.llm/prompts/system.md`** — там описаны все правила, структура папок и твоя роль.
2.  **Загрузи контекст последних изменений:** Найди самый свежий файл в `.llm/summary/` и ознакомься с ним.
3.  **Не выполняй поиск по всему Vault**, пока не поймешь текущую задачу из Summary.
4.  **Соблюдай запрет на чтение архивов**, описанный в `system.md`.

*Для ознакомления с игровыми событиями жди команды: «Ознакомься с последней сессией».*
