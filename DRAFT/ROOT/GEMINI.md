# GEMINI.md

Инструкции для Gemini при работе с проектом Val'trey D&D.

---

## ОБЯЗАТЕЛЬНАЯ ИНИЦИАЛИЗАЦИЯ

**КРИТИЧЕСКИ ВАЖНО:** При каждой новой сессии прочитай:

```
C:\GoogleDrive\DND\.llm\prompts\system.md
```

Этот файл содержит **полный контекст сеттинга**, правила навигации, работы с файлами, специализированные задачи (skills) и всю необходимую информацию для работы.

**Без прочтения `system.md` твоя инициализация НЕ ЗАВЕРШЕНА.**

---

## Специфика для Gemini

**Основные задачи:**
- Генерация изображений по промптам
- Анализ видео/аудио записей игровых сессий

**Skills:**
- Для генерации изображений используй: `.llm/skills/image_prompt.md`
- Для обработки сессий: `.llm/skills/session_processing.md`

---

## Документирование

При работе с проектом:
- **Changelog:** Помечай изменения как `(gemini)`
- **Резюме:** Используй формат `gemini_{YYYY-MM-DD}.md` в `.llm/summary/`
- **Лог изменений:** Записывай в `.llm/log/changelog.md` с пометкой `### gemini`
