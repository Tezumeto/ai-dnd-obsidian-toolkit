# Skill: Создание магических предметов

## Когда использовать
- Пользователь просит создать магический предмет, артефакт, экипировку
- Награда за квест или данж
- Уникальный предмет для NPC
- Легендарный артефакт для лора
- Homebrew зелье, свиток, кольцо, оружие, доспех, чудесный предмет

---

## Контекст (что читать)

### Обязательно:
1. **Уровень партии** — определяет допустимую редкость (см. «Редкость по уровням»)
2. **Шаблон предмета:** `Obsidian/DnD.Valtray/zUtilities/Templates/ITEM Template.md`

### Опционально:
- Магические системы сеттинга — если есть специфика
- История/NPC — если артефакт связан с персонажем или событием
- Состав партии — чтобы предмет не дублировал классовые способности

---

## Workflow

### Шаг 1: Интервью — сбор информации

Не задавай все вопросы сразу — группируй логически. Если пользователь уже описал идею, извлеки ответы и уточни только недостающее.

**Группа 1 — Основа:**
1. **Название** — на русском, можно и на английском
2. **Тип предмета** — оружие, доспех, щит, зелье, кольцо, жезл, посох, свиток, палочка, чудесный предмет
3. **Подтип** — если оружие: длинный меч, боевой топор и т.д. Если доспех: кожаный, латный и т.д.
4. **Редкость** — common, uncommon, rare, very rare, legendary, artifact
5. **Основной эффект** — что предмет делает?

**Группа 2 — Механика:**
6. **Настройка** — требуется ли? Есть ли ограничения (класс, мировоззрение, раса)?
7. **Заряды** — есть ли? Сколько? Как восполняются? Что при 0?
8. **Стоимость** — примерная цена или «бесценный»
9. **Для кого** — уровень партии, конкретный получатель?

**Группа 3 — Лор:**
10. **Происхождение** — кто создал, когда, при каких обстоятельствах?
11. **Предыдущие владельцы** — были ли?
12. **Где находится** — кто носит, где спрятан?

**Группа 4 — Расширенное (опционально):**
13. **Проклятие** — есть ли проклятие или негативный эффект?
14. **Секретные свойства** — скрытые эффекты?
15. **Связи** — связь с другими предметами, NPC, организациями?
16. **Визуальное описание** — материалы, цвета, руны, свечение?
17. **Сюжетные крючки** — какие квесты может запустить?

Если пользователь говорит «сделай на своё усмотрение» — заполни пропуски, ориентируясь на тон сеттинга (если известен) или на классическое D&D фэнтези.

### Шаг 2: Проверка по принципам дизайна

Перед созданием пройдись по принципам и RANSAC (см. разделы ниже). Убедись, что предмет не нарушает ни одного принципа.

### Шаг 3: Заполнение шаблона

Прочитай шаблон из `Obsidian/DnD.Valtray/zUtilities/Templates/ITEM Template.md`. Заполни все поля, руководствуясь правилами из раздела «Справочник по полям» ниже.

### Шаг 4: Сохранение

**Вариант A — Obsidian MCP (приоритетный):**
Спроси папку для сохранения, запиши через `obsidian:write_note` / `obsidian:write_file`.

**Вариант B — Файл для скачивания:**
Создай .md файл, покажи пользователю.

### Шаг 5: Финальный вывод

```
✅ Предмет создан: {Название}
📋 Тип: {тип}, {редкость}
⚡ Ключевое свойство: {основной эффект в одном предложении}
🖼️ Промпт для изображения готов (--ar {ratio})
📁 Сохранён: {путь}
```

## Принципы дизайна

### Принцип 1: Одна главная фишка на предмет

Каждый предмет должен делать **одну вещь хорошо**. Несколько мощных эффектов — это Rare+ или плохой дизайн.

> ✅ Хорошо: плащ даёт +1 КД и +1 к спасброскам — одна простая тема
> ❌ Плохо: меч даёт +2 к атаке, 2d6 огня, летит сам, регенерирует ХП

### Принцип 2: Не перекрывай классовые способности

Предмет должен **дополнять** класс, а не заменять его сильнейшие черты.

> ❌ Плохо: плащ с постоянной Невидимостью делает «Прятаться» бесполезной
> ✅ Хорошо: тот же плащ с Невидимостью 1/день — усиление, а не замена

**Вопрос-тест:** *«Обесценивает ли предмет что-то, чему персонаж учился несколько уровней?»*

### Принцип 3: Предмет рассказывает историю

Лучшие предметы имеют нарратив: кто сделал, зачем, что произошло. Механика и флёр должны быть связаны.

> ✅ Хорошо: «Амулет позволяет говорить с мёртвыми, но стирает воспоминания. Создан некромантом, который сам забыл, зачем его сделал.»
> ❌ Плохо: «Магический меч +2. Сделан гномом.»

### Принцип 4: Ограничения делают предмет интереснее

Хорошие ограничения: заряды (1–3/день), attunement, условие активации (только в темноте, против конкретного типа врага), цена (усталость, потеря ХП), ограничение по классу/расе.

### Принцип 5: Шкала «функциональный → захватывающий»

| Тип эффекта | Пример | Уровень интереса |
|-------------|--------|-----------------|
| Бонус к числам | +1 к атаке | Полезно, но скучно |
| Новый ресурс | 1d4 зарядов заклинания | Лучше |
| Новая способность | Дышать под водой | Интересно |
| Меняет стиль игры | Все атаки по тебе с помехой | Захватывающе |
| Меняет взаимодействие с миром | Говорить с любым животным | Незабываемо |

Стремись к «новой способности» и выше. Чистые бонусы к числам — только как второстепенный элемент.

## RANSAC — чеклист механики

Перед финализацией любого предмета:

| Буква | Вопрос |
|-------|--------|
| **R** — Rarity | Редкость соответствует силе? |
| **A** — Attunement | Нужна ли настройка? Если мощный — да |
| **N** — Negation | Что отключает предмет? (антимагическое поле, условия) |
| **S** — Save | Если есть эффект на врага — какой спасбросок и DC? |
| **A** — Attack | Есть ли бросок атаки? Ближний, дальний, заклинание? |
| **C** — Charges | Сколько зарядов, как восстанавливаются, что при 0? |

## Редкость по уровням

| Редкость | Уровни | Суть | Attunement |
|----------|--------|------|-----------|
| **Common** | 1–4 | Одна мелкая способность, не влияет на бой | Обычно нет |
| **Uncommon** | 1–5 | Одна серьёзная способность 1–2/день **или** постоянный ситуативный бонус | Иногда |
| **Rare** | 5–10 | Одна мощная постоянная способность **или** несколько средних | Почти всегда |
| **Very Rare** | 11–16 | Несколько мощных способностей, заклинания 5–6 круга | Да |
| **Legendary** | 17+ | Уникальные способности, сюжетный вес | Да |
| **Artifact** | Любой | Сила, меняющая кампанию, всегда с ценой | Да |

> ⚠️ Rare на уровнях 3–4 допустимо как особая находка, но это **аванс**. Не несколько Rare у одного персонажа.

## Официальные ориентиры

### Uncommon — эталоны:
| Предмет | Механика |
|---------|---------|
| Bag of Holding | Бездонная сумка (постоянно, без attunement) |
| Cloak of Protection | +1 КД и ко всем спасброскам (attunement) |
| Goggles of Night | Darkvision 60ft (постоянно) |
| Boots of Elvenkind | Преимущество на Скрытность при движении |
| Hat of Disguise | *Disguise Self* по желанию |
| Winged Boots | Полёт 4 часа/день (attunement) |

**Паттерн Uncommon:** постоянный ситуативный бонус **или** слабый эффект без ограничений **или** сильная способность 1/день.

### Rare — эталоны:
| Предмет | Механика |
|---------|---------|
| Amulet of Health | CON = 19 (attunement) |
| Bracers of Defense | +2 КД без брони и щита (attunement) |
| Cloak of Displacement | Все атаки по тебе с помехой до первого попадания (attunement) |
| Necklace of Fireballs | 1–7 шаров 8d6 (расходники) |
| Gem of Seeing | *True Seeing* 3/день (attunement) |

**Паттерн Rare:** постоянный мощный эффект **или** сильная активная способность несколько раз в день **или** комбинация средних эффектов.

## Дизайн зарядов

| Частота | Сила эффекта | Редкость |
|---------|-------------|---------|
| Безлимитно | Слабый (кантрип) | Common |
| 1/день | Средний (заклинание 1–2 круга) | Uncommon |
| 2–3/день или 1d3 восст. | Средний-сильный | Uncommon–Rare |
| 1/день | Мощный (заклинание 3–4 круга) | Rare |
| Постоянно | Слабый пассив | Uncommon |
| Постоянно | Мощный пассив | Rare–Very Rare |

Всегда прописывай что происходит при 0 зарядов.

---

## Типичные ошибки

- ❌ **Слишком много эффектов** — три мощных способности без Legendary-редкости
- ❌ **Нет ограничений** — безлимитный мощный эффект
- ❌ **Перекрывает класс** — плащ невидимости убивает Плута
- ❌ **Нет флёра** — чисто механический предмет без истории
- ❌ **DC не задан** — «враг проходит спасбросок» без DC и типа
- ❌ **Тип спасброска не указан** — всегда указывай STR/DEX/CON/INT/WIS/CHA
- ❌ **Восстановление не задано** — «3 заряда» без указания когда
- ❌ **Слишком ситуативно** — работает только в одном крайне редком условии

## Справочник по полям

### Типы предметов (item_type)

`оружие` · `доспех` · `щит` · `зелье` · `кольцо` · `жезл` · `посох` · `свиток` · `палочка` · `чудесный предмет`

### Подтипы оружия (item_subtype)

**Простое рукопашное:** дубинка, кинжал, палица, копьё, лёгкий молот, ручной топор, метательное копьё, булава, боевой посох, серп.
**Простое дальнобойное:** лёгкий арбалет, короткий лук, дротик, праща.
**Воинское рукопашное:** боевой топор, боевой молот, длинный меч, моргенштерн, рапира, секира, глефа, алебарда, короткий меч, трезубец, цеп, пика.
**Воинское дальнобойное:** длинный лук, тяжёлый арбалет, ручной арбалет.

### Подтипы доспехов (item_subtype)

**Лёгкие:** стёганый, кожаный, проклёпанный кожаный.
**Средние:** шкурный, кольчужная рубаха, чешуйчатый, нагрудник, полулаты.
**Тяжёлые:** кольчужный, наборный, латный.

### Стоимость по редкости

| Редкость | Стоимость | Категория |
|----------|----------|-----------|
| Common | 50–100 зм | дешёвый |
| Uncommon | 101–500 зм | дорогой |
| Rare | 501–5 000 зм | очень дорогой |
| Very Rare | 5 001–50 000 зм | очень дорогой |
| Legendary | 50 001+ зм | бесценный |
| Artifact | — | бесценный |

### Настройка (attunement_requirement)

`""` (нет ограничений) · `"заклинатель"` · `"волшебник"` / `"жрец"` / `"колдун"` и т.д. · `"добрый"` / `"злой"` · конкретная раса

### Формулы зарядов (charges_recharge)

`"рассвет 1d6+1"` · `"рассвет 1d4"` · `"рассвет все"` · `"закат 1d6"` · `"не восполняется"` · `"особое"` · `"при 0 бросок d20"` (на 1 — уничтожается)

### Свойства оружия

фехтовальное · лёгкое · тяжёлое · досягаемость · метательное · двуручное · универсальное · боеприпас · перезарядка · особое

### Соотношение сторон для изображений

| Тип предмета | --ar | Почему |
|-------------|------|--------|
| Мечи, посохи, копья, жезлы, алебарды | 9:16 | Вытянутые вертикально |
| Кольца, амулеты, зелья, мелкие предметы, щиты | 1:1 | Квадрат для детализации |
| Плащи, ковры, широкие предметы | 16:9 | Горизонтальные |
| Полные доспехи | 9:16 | Вертикальная фигура |
| Книги, шкатулки | 1:1 | Компактные |
| Если непонятно | 1:1 | Безопасный вариант |

### Статусы предметов (status)

`active` · `lost` · `hidden` · `draft` · `archived`

## Правила заполнения шаблона

### Frontmatter

- **attunement**: `true` или `false` (без кавычек)
- **charges**: число или `0`
- **save_dc**: число или `0`
- **weight**: число в фунтах
- **tags**: всегда включай `magic-item`, редкость, тип предмета и тематические теги

### Блоки свойств

Описание свойств пиши в стиле официальных книг D&D 5e (но на русском): сначала общее описание, потом детали по каждому свойству.

**Заряды** (если есть):
```
Предмет имеет **N зарядов**. Вы можете потратить заряды:
- **Название (X зарядов).** Описание.
**Восстановление.** Формула на рассвете. При 0 — бросок d20, на 1 теряет свойства *(если применимо)*.
```

**Заклинания** (если даёт):
```
Находясь настроенным, вы можете сотворить (DC N):
- **Заклинание** (X зарядов)
```

**Проклятие** (если есть):
```
*Скрыто до срабатывания или успешной проверки Магии / заклинания Идентификация.*
Описание...
```

### Промпт для изображения

Генерируй на английском. Структура:
```
[item name], [visual description], [materials, colors, runes, glow], [background: dark surface / stone altar / wooden table / velvet cloth], style: dnd art, digital art, detailed illustration, dramatic lighting --ar [RATIO]
```

Стиль визуала подбирай под атмосферу сеттинга пользователя. По умолчанию: фэнтези, приглушённые тона с магическими акцентами.

### Карточка для игрока

ТОЛЬКО информация, доступная при идентификации/настройке. Никаких проклятий, секретов, сюжетных крючков.

### Заметки мастера

- **Истинное назначение** — роль в сюжете
- **Секретные свойства** — то, чего нет в карточке игрока
- **Сюжетные крючки** — 2–3 идеи квестов
- **Баланс** — не слишком ли силён/слаб для целевого уровня

## Примеры homebrew-предметов по редкостям

### Common: Светляк Путника
```
Светляк Путника
Чудесный предмет (фонарь), common

Небольшой каменный шар, умещающийся в ладони. Испускает тусклый свет
в радиусе 10 футов. Свет не гасится ветром и водой. Бонусным действием
можно приглушить до полного погасания и зажечь снова.
```

### Uncommon: Коралловый Клинок
```
Коралловый Клинок
Оружие (длинный меч), uncommon (requires attunement)

+1 к броскам атаки и урона. Пока держишь меч — можешь дышать под водой.
Лезвие покрыто живым кораллом, который медленно растёт и меняет форму.
```

### Rare: Амулет Забытых Голосов
```
Амулет Забытых Голосов
Чудесный предмет (амулет), rare (requires attunement)

1/день — заклинание Говорить с мёртвыми (DC 15). После использования:
спасбросок Мудрости DC 12, при провале — забываешь случайное событие
последних 24 часов (на усмотрение DM).

Проклятие. Амулет не снимается без Снятие проклятия. Каждую полночь:
спасбросок Мудрости DC 10 или потеря случайного воспоминания.
```

## Финальный чеклист

- [ ] Редкость соответствует уровню партии
- [ ] Одна главная способность (не три сразу)
- [ ] Не перекрывает классовые способности
- [ ] RANSAC пройден (Rarity / Attunement / Negation / Save / Attack / Charges)
- [ ] Заряды: количество + восстановление + поведение при 0
- [ ] Флёр: история, внешний вид, создатель
- [ ] Frontmatter заполнен
- [ ] Wikilinks расставлены
- [ ] Промпт для изображения сгенерирован
- [ ] Карточка для игрока без спойлеров
