# Структура проекта Val'trey

Проект разделён на **лор** (статичные материалы сеттинга) и **операционную часть** (текущие игры).

## Пути

Рабочая директория Claude Code: `I:\Мой диск\DND\`
Vault Obsidian: `I:\Мой диск\DND\Obsidian\DnD.Valtray\`

При работе с файлами учитывай, какой инструмент используешь:
- **Grep / Glob / Read / Edit** — нужен полный путь от CWD: `I:\Мой диск\DND\Obsidian\DnD.Valtray\0.Лор\...`
- **MCP Obsidian** (read_note, search_notes, write_note) — нужен путь относительно Vault root: `Obsidian/DnD.Valtray/0.Лор/...`

## Принципы навигации по папкам

- **`0.Лор/`** — источник истины о мире. Изменяй осторожно, проверяя на противоречия.
- **`Сессии/`** — оперативные данные текущих игр. Можно менять свободнее.
- **`Приключения/`** — готовые модули. При изменении проверяй связность.
- **`1.Черновик/`** — рабочая область для экспериментов.
- **`zАрхив/DELETED/`** — сюда перемещай удаляемые файлы.
- **`zАрхив/`** — не читай без явной просьбы (устаревшая информация).
- **`.llm/log/`** — только для записи, не читай без просьбы (экономия контекста).
